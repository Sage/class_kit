# ClassKit

Welcome to your ClassKit! ClassKit is a toolkit for working with entities & classes.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'class_kit'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install class_kit
